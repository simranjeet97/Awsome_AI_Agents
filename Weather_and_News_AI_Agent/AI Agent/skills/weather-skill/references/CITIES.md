# City Name Aliases

Some city names have common alternate spellings or historical names.
When the user provides one of these, silently use the correct name instead.

|----------------|------------------------|
| Bangalore      | Bengaluru              |
| Bombay         | Mumbai                 |
| Calcutta       | Kolkata                |
| Madras         | Chennai                |
| Peking         | Beijing                |
| Canton         | Guangzhou              |
