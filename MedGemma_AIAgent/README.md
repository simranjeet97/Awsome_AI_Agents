# 🩺 MedGemma Medical Scan Analyzer

An advanced AI-powered medical scan analysis platform built with **MedGemma 1.5-4B** and **Streamlit**. This tool provides clinical-grade multimodal reasoning for CT, MRI, and Mammography scans with a built-in **Human-in-the-Loop (HITL)** workflow.

## 🚀 Key Features

-   **Multimodal Reasoning**: Utilizes Google's MedGemma 1.5-4B model for specialized medical visual question answering and analysis.
*   **Broad Format Support**: Accepts medical formats (DICOM, NIfTI) and standard image formats (JPG, PNG, WebP, TIFF).
*   **Clinical Windowing**: Optimized preprocessing pipeline that maps medical intensity windows (Bone, Soft Tissue, Lung) to RGB channels for enhanced AI perception.
*   **Human-in-the-Loop Workflow**: Clinicians can review, edit, and validate AI-generated findings before final report generation.
*   **Professional Reporting**: Generates validated PDF clinical reports with digital signatures.
*   **Clean Clinical UI**: High-contrast, accessibility-focused interface designed for medical professionals.

## ⚙️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/simranjeet97/Awsome_AI_Agents.git
   cd Awsome_AI_Agents/MedGemma_AIAgent
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment**:
   Create a `.env` file and add your Hugging Face access token:
   ```env
   HF_TOKEN=your_huggingface_token_here
   ```

## 📖 Usage

1. **Start the application**:
   ```bash
   streamlit run app.py
   ```

2. **Upload Scans**: Drag and drop any medical study (DICOM/NIfTI) or image (JPG/PNG).
3. **Configure Analysis**:
   - For real AI analysis, uncheck **Demo Mode** in the sidebar.
   - For UI testing, keep **Demo Mode** enabled.
4. **Review & Sign-off**: Edit the AI's "first draft" findings and enter the clinician's signature to generate a PDF report.
