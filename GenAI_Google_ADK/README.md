# 🛠️ Google ADK Showcase & Toolkit

A comprehensive resource for learning and building with the **Google Agent Development Kit (ADK)**. This folder contains a progression of projects—from beginner-level search tools to a full-stack research application—demonstrating how to leverage Google's agentic ecosystem.

---

## 📂 Folder Structure

This toolkit is organized into three levels of complexity:

1.  **[GoogleADK_Beginner](./GoogleADK_Beginner):**
    - **Focus:** Fundamental agent creation and structure.
    - **Outcome:** Minimalist "Hello World" style agents using Gemini and ADK.

2.  **[GoogleADK_Search](./GoogleADK_Search):**
    - **Focus:** Tools and Grounding.
    - **Outcome:** Agents integrated with Google Search to provide up-to-date, grounded information.

3.  **[GoogleADK_App](./GoogleADK_App):**
    - **Focus:** Full Application Integration.
    - **Outcome:** A cohesive application structure combining frontend, backend, and multi-agent workflows.

---

## 🛠️ Tech Stack

- **Model:** Gemini 2.0 / 1.5 Flash
- **Framework:** [Google Agent Development Kit (ADK)](https://cloud.google.com/vertex-ai/docs/generative-ai/agent-engine/overview)
- **Infrastructure:** Vertex AI, Google Cloud Platform
- **Integration:** Google Search Tooling

---

## 🌟 Why Google ADK?

Google ADK provides a robust backbone for building enterprise-grade agents with:
- **Managed Runtime:** Easy deployment via Google Cloud.
- **Native Grounding:** Seamless integration with Google Search and enterprise datasets (Vertex AI Search).
- **Control & Scalability:** Built-in support for orchestration and tool-calling with high reliability.
